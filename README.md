## Brain Games 

- I built Brain Games, a 5-level interactive web experience wehre players must complete every game in order to unlock a surpise! The games are: Jeopardy, Hangman, Math Spring, Memory Game, and a final Riddle challenge. 

- How to Play: 
    - Jeopardy: Choose 9 topics and answer correctly. Get at least 3 right to move on
    - Hangman: A cowboy-themed word guessing game. Guess the word or the board resets and keeps going until you get it
    - Math Sprint: You have 60 seconds to answer math questions as fast as possible. Answer correctly to level up; get one wrong and you level down (and lose 5 seconds)
    - Memory Game: Flip cards to find matching pairs. Every 2 wrong guesses adds more cards to the board. Your goal: finish as fast as possible 
    - Riddle: Solve the final riddle to unlock the surprise 

- Complete all 5 levels and earn a downloadable graduation diploma as your reward! 
- Don't want to play in order? Hit "More Games" from the main menu to jump directly into any game you'd like! No levels, no pressure, just fun! 

- Live Site: https://julisam101.github.io/braingames/

- What I learned: 
    - How to properly incorporate and manipulate APIs to power game content
    - How to apply unique themes to each game to make the expereince more immersive and interactive
    - How to use JavaScript conditions to set up different paths and outcomes based on player choices
    - How to generate a personalized, downnloadable graduation diploma as a PDF reward
    - How to build a "More Games" section that lets players jump to any game without going through the levels
    - How to reset game state without requiring a full page refresh
